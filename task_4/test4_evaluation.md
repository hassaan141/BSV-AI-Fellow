# Test 4 Evaluation: Iterative Research Loop

## What Is Test 4?

Test 4 checks whether Mem0 memory compounds over multiple research loops.

The flow is:

1. Clear memory.
2. Run Loop 1 with Nemotron on VLA foundations.
3. Run Loop 2 with Nemotron on world models.
4. Run Loop 3 with Kimi to synthesize across prior memory.

The core question:

> Does memory become a compounding research asset, or does it just accumulate a loose pile of facts?

## What Are We Trying To Learn?

We are testing whether later research loops improve because of earlier stored memories.

Specifically:

- Does Loop 2 use Loop 1's memories?
- Does Loop 2 add genuinely new information?
- Does Kimi synthesize across both loops?
- Do stored memories stay specific and useful?
- Do duplicates or contradictions accumulate?

## Evaluation Metrics

| Metric | What To Check |
|---|---|
| Memory growth | How many memories exist after each loop? |
| Retrieval quality | Did later loops retrieve relevant prior memories? |
| Compounding | Did later outputs build on earlier outputs? |
| Novelty | Did each loop add new information? |
| Synthesis | Did Kimi connect facts into a useful investment/product view? |
| Duplication | Did memories repeat the same claims? |
| Specificity | Were memories concrete or vague? |
| Net workflow value | Was this better than one long prompt? |
